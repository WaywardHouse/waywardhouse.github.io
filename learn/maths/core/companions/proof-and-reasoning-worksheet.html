Use This Worksheet

This companion sheet is built for handwritten work on an iPad or on paper. The point is not to copy the final polished proof immediately. The point is to force the structure of the reasoning into the open.

Try each problem in this order:

  1. Decide what must be assumed and what must be shown.
  2. Pick the proof method before you start writing.
  3. Write the key definitions explicitly.
  4. Draft the argument in the working space.
  5. Rewrite the clean version in the final-proof box.

Use the chapter page only after attempting the proof yourself.
